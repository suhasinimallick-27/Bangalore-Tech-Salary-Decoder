python and Pandas project for analyzing Bangalore tech salaries and finding salary insights.

Bangalore Tech Salary Decoder

The Bangalore Tech Salary Decoder is a data analytics project that analyzes salary patterns of 1,000 Bengaluru tech professionals. The project focuses on cleaning messy salary data, standardizing job roles, and extracting meaningful insights related to experience, skills, company type, and compensation.

The dataset contains information about job roles, years of experience, current salary, previous salary, company type, skills, education tier, and work mode. Since the dataset contains inconsistent formats and missing values, data cleaning techniques were applied before analysis.

The project answers important business questions such as:

- Which tech roles receive the highest salaries?
- How does experience affect salary growth?
- Which skills provide the highest salary premium?
- How do company types influence compensation?
- Who are the most underpaid professionals?

Technologies Used:

- Python
- Pandas
- NumPy
- Google Colab - https://colab.research.google.com/drive/1bbK5Hph7YjnUN8p6lBGZXYcmtrdpG8e9?usp=sharing

Key Features:

- Data cleaning and preprocessing
- Salary conversion and standardization
- Role-based salary analysis
- Experience-based salary growth analysis
- Skill premium analysis
- Company type comparison
- Underpaid employee identification
- Final printed salary report

This project was completed as a 2-hour live industry sprint under Unlox Academy and demonstrates practical data analysis skills using Python fundamentals and Pandas.
